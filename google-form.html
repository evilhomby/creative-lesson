    # 借用 Google 表單 + 自訂樣式

    Google 表單提供了許多工具方便快速建立表單，並且將回覆結果記錄試算表中。可以視為一個完整的小形資料庫，對於前端設計師來說是非常方便的服務。我們可以透過 Google 表單建立，再加上自訂的樣式來完成專案。

    例如設計一個報名網站。我們可能需要的欄位有：

    • 姓名
    • 手機

    # 前置作業：使用 Google Form 建立表單

    快速的在 Google Form 上建立預期收集資料欄位的表單，完成後使用 Chrome 的開發者工具檢視表單，複製 <form> 的 action 屬性與各表單欄位的 name 屬性。

    開啟原始碼，搜尋 action=" 與 name="entry 可以快速找到需要的屬性

    # 使用 HTML Form 的方式傳送資料

    自行設計一個表單與樣式，並將前述的屬性值填在對應的要素上。

    如 Google Form 上有設定驗證，自行設計的表單需對應製作驗證才能直接將結果寫入回應試算表；否則會跳轉到 Google From 上。

    前端可以使用 pattern, maxLength 進行表單驗證；Google 表單設定的驗證，如：必填、特殊格式可視為後端驗證。前後端驗證一致可避免表單資料未寫入回應試算表卻跳轉自訂成功頁面的狀況。

    # 範例 & 實作

    使用 Google 內建提交成功

    <form action="https://docs.google.com/forms/d/e/1FAIpQLSe0ByMAnRzvbEcbyQeVc0aQMsNOR9PXLMVNtUSZvMd5gFTn_g/formResponse" method="post">
	<label>姓名：<input name="entry.860713921"></label><br>
	<label>手機：<input name="entry.61241330"></label><br>
	<input type="submit" value="送出">
</form>


    使用自訂提交成功頁面或隱藏

    • 表單設定 target 屬性，資料送出指向一個隱藏的 <iframe> 並透過 js 控制載入後跳轉指定頁面功能。
    • <form> target 的值要與 <iframe> 的 name 相符
    • 修改 onload="..." 中的網址即為自訂轉向目的 (此行是 js 程式請先複製貼上使用)
    <form action="https://docs.google.com/forms/d/e/1FAIpQLSe0ByMAnRzvbEcbyQeVc0aQMsNOR9PXLMVNtUSZvMd5gFTn_g/formResponse" method="post" target="custom">
	<label>姓名：<input name="entry.860713921"></label><br>
	<label>手機：<input name="entry.61241330"></label><br>
	<input type="submit" value="送出">
</form>
<iframe style="display:none;" name="custom" onload="this.onload=function(){window.location='https://www.e-stork.com.tw'}"></iframe>


    # 使用 jQuery Ajax 的方式 (快速帶過，以後就會懂了)

    以 jQuery 為例，首先需要載入函式庫，並在 Document Ready 後執行

    <script src="http://code.jquery.com/jquery-1.8.3.min.js"></script>
<script>
$(function() {
	$('form').submit(function() {
		$.post(url, {
				'name-123456789': $('[name="name-123456789"').val(),
				'mobile-123456789': $('[name="mobile-123456789"').val()
			}, function() {
			console.log('表單已送出');
		});
	});
})
</script>
