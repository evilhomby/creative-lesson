    # SublimeServer 套件

    靜態網頁開發使用瀏覽器直接開啟 html 檔即可進行瀏覽，當然也可以在伺服器環境下進行相同的測試。

    而透過 JavaScript 或 jQuery, Vue 等函式庫 ajax 載入資料的情況，則需要伺服器環境來進行測試。

    SublimeServer 套件可簡易快速的建立伺服器環境供測試

    # 安裝與使用步驟

    • Package Control 中輸入 Install Package
    • 搜尋 SublimeServer 並安裝
    • 於選單列 Tools > SublimeServer > start SublimeServer 啟用
    • 建立一個資料夾並加到專案中 (Project > Add Folder to Project)
    • 將 html 檔儲存在專案資料夾中，按右鍵並選擇 View in SublimeServer

    # 在 Mac 中查詢 IP

    • 偏好設定 > 網路中即可查看
    • 或透過終端機輸入 ifconfig 亦可 (if = interface 的縮寫)

    # 共同檢視專案

    將查詢到的 IP 後方加上 Port (預設是 8080)，即可連到伺服器進行網頁檢視。

    # 備註
