    # 關於 JavaScript 的日期、時間

    var now = new Date();
console.log(now);

    使用 getFullYear(), getMonth() 等方法取得日期時間資訊。

    使用 setFullYear(), setMonth() 等方法可以改變日期物件資訊。

    其中需要注意的是月份為 0-11 計算 (1 月 - 12 月)；星期則是 0-6 (星期日 - 星期六)

    相關的方法可參考 W3School Date

    //顯示今日日期
var today = new Date();
console.log(today.getFullYear(), today.getMonth() + 1, today.getDate());

//顯示明日日期 (日期的增減)
var tomorrow = new Date(today.getFullYear(), today.getMonth(), today.getDate() + 1);
console.log(tomorrow.getFullYear(), tomorrow.getMonth() + 1, tomorrow.getDate());

//產生一個指定日期
var custom1 = new Date(2019, 5, 20);
var custom2 = new Date('2019-06-20T00:00:00');

//比較日期是否相同要使用 getTime()
console.log(custom1 == custom2); //false
console.log(custom1.getTime() == custom2.getTime()); //true

    # 計時事件

    有 setTimeout(fn, ms) 與 setInterval(fn, ms) 兩個函式，差別在於執行的次數。

    setTimeout 會在指定的毫秒執行 1 次

    // 3 秒後在主控台顯示訊息
var timeout = setTimeout(function() {
	console.log('hello world');
}, 3000);

//未執行前可以使用 clearTimeout() 來取消
clearTimeout(timeout);

    而 setInterval 則是間隔指定的毫秒執行，一直執行到明確由程式要求停止。

    //每隔一秒在主控台顯示訊息
var sum = 0;
var interval = setInterval(function() {
	console.log(sum++);
}, 1000);

//使用 clearInterval() 停止計時事件
clearInterval(interval)

    以 setTimeout 做 setInterval 相同的事，自己呼叫自己的概念。

    var timeout;
function clock() {
	timeout = setTimeout(function(){
		clock();
	}, 1000);
	console.log(new Date().getSeconds())
}
clock();
