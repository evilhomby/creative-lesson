    # 課程進度測驗

    # 1. 畫面上產生 n 個方塊

    可自行建立 HTML 結構，或以 JavaScript 動態產生

    排列為矩陣可透過以下幾種方式：

    • 父層容器設定 display:flex;flex-wrap:wrap;
    • 要素本身設定 float (記得父層要修正高度)
    • 要素本身設定 display:inline-block (但手動產生的情況要素間會出現空白，需透過調整要素換行解決)
    <div id="container"></div>
<script>
var blockNum = 25;
var container = document.querySelector('#container');
for(var i = 0; i < blockNum; i++) {
	//注意，空要素在畫面上看不到，需給予內容或透過 CSS 給予高度
	var div = document.createElement('div')
	div.classList.add('block');
	div.innerHTML = i; //使用 innerHTML 給予內容
	container.append(div);
}
</script>

    # 3. 隨機挑選一個方塊的透明度為 0.85-0.95 之間的小數

    取 0.85-0.95 之間的小數，可先取 85-95 的整數再除以 100 來取得

    (Math.floor(Math.random() * (95 - 85 + 1)) + 85) / 100;

    隨機取得特定的某個方塊，可以使用 length 屬性

    var blocks = document.querySelectorAll('.block');
var randBox = Math.floor(Math.random() * blocks.length);

    # 開發習慣建議

    • 英打與常用英文單詞請多加練習與記憶
