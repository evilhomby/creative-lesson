    # 動畫效果的函式庫: TweenMax/TweenLite

    課程推薦使用的是 Green Sock 的 TweenMax/TweenLite，除了可以控制要素、值的漸變運動，也有時間軸可做進階動畫。以下以 TweenMax 進行說明。

    # 使用方式

    # 載入函式庫

    • 透過 CDN (內容傳遞網路) 將函式庫檔案載入
    • 自行下載 js 檔並於網頁中載入
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/2.0.2/TweenMax.min.js"></script>

    # 在 Docuemt Ready 完成後執行動畫相關程式

    Document Ready 事件發生代表瀏覽器已經將 HTML 解析完畢，準備渲染到畫面上。

    為什麼要在 Document Ready 後才能執行呢？是因為若在這個事件之前執行，可能會發生找不到對象的錯誤。

    # 使用 TweenMax.to() 製作動畫

    TweenMax.to() 是指由要素原有的狀態漸變至 to() 中指定的狀態。反之也有 TweenMax.from() 可以使用。

    例如：要素坐標的移動

    TweenMax.to(el, sec, {x:100, y:100});
