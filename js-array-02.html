    # 陣列元素的增減

    • unshift(a, b, ..., n)：加到最前面
    • push(a, b, ..., n)：加到最後面
    • shift()：刪除最前一個
    • pop()：刪除最後一個
    • splice(start, n[, item])：增加或刪除元素
    var arr = [1, 2, 3, 4, 5];
arr.unshift(0); // [0, 1, 2, 3, 4, 5]
arr.push(6, 7, 8); // [0, 1, 2, 3, 4, 5, 6 ,7 ,8]
arr.shift(); // [1, 2, 3, 4, 5, 6 ,7 ,8]
arr.pop(); // [1, 2, 3, 4, 5, 6 ,7]
arr.splice(2, 3); // [1, 2, 6, 7]
arr.splice(2, 0, 'x'); // [1, 2, 'x', 6, 7]

    # 陣列的排序

    將陣列反轉排序使用 reverse() 即可。

    var arr = [1, 2, 3, 4, 5];
arr.reverse();

    依大小排序使用 sort() 方法，但需注意預設是轉換為字串後依 charCode 由小排到大，因此數字排序需自定函式。

    var arr = [1, 10, 30, 2, 20, 3];
arr.sort(); //  [1, 10, 2, 20, 3, 30]

arr.sort(function(a, b) {
	return a - b; // [1, 2, 3, 10, 20, 30]
});

    英文字母的排序是依 charCode 由小到大排序，可使用 String 物件的 charCodeAt() 來得到 charCode。

    var arr = ['a', 'A', 'B', 'b'];
arr.sort(); // ['A', 'B', 'a', 'b']
console.log(arr);

arr.forEach(function(v) {
	console.log('%s 的 charCode 是 %s', v, v.charCodeAt(0));
});

    預設是由小到大排序，想要改為由大到小怎麼辦？

    • 先由小到大排序再反轉
    • 自訂排序函式

    # 輪詢陣列

    可以使用 forEach() 或 for 迴圈來輪循整個陣列, 代入一個匿名函式，第一個參數是值 (value)，第二個參數是索引 (index)。 通常會以 v 與 i 來表示，但可以更改為明確的單詞。

    var arr = [1, 2, 3, 4, 5];
arr.forEach(function(v, i) {
	console.log('arr[%s] = %s', i, v);
});

var arr = [1, 2, 3, 4, 5];
for(var i = 0; i < arr.length; i++) {
	console.log('arr[%s] = %s', i, arr[i]);
}
    # 輪詢二維陣列

    匿名函式中的參數建議使用明確的單詞，以避免混淆。

    var fruits = [
	['banana', 20],
	['apple', 30],
	['cherry', 50]
];
fruits.forEach(function(fruit, i) {
	//fruit = fruits[i] 撰寫時可選用前者較為精簡
	fruit.forEach(function(item, j) {
		console.log('fruits[%s][%s] = %s', i, j, item);
	});
});

    # 映射陣列

    Map (映射)的概念是把陣列 A 依照一個規則 (也可以沒有) 逐個處理後回傳為陣列 B。

    例如：將一個陣列中的值變成該值的平方

    var arr = [1, 2, 3, 4, 5];
var res = arr.map(function(v, i) {
	return v * v;
});
console.log(arr, res);

    當然妳也可以這樣做

    var arr = [1, 2, 3, 4, 5];
var res = [];
arr.forEach(function(v, i) {
	res.push(v * v);
});
console.log(arr, res);

    # 過濾陣列

    var arr = [1, 2, 3, 4, 5];
var res = arr.filter(function(v, i) {
	return v % 2 == 0;
});
console.log(arr, res);

    var arr = [1, 2, 3, 4, 5];
var res = [];
arr.forEach(function(v, i) {
	if(v % 2 == 0) {
		res.push(v);
	}
});
console.log(arr, res);
    # 使用時機

    forEach, map, filter 的寫法非常類似，使用時機的判斷如下：

    • forEach 可處理 map, filter 相同的事，如果不確定，可無條件選用。
    • map 用於陣列值處理並回傳為一個新的陣列的情況。
    • filter 用於過濾陣列為一個新的陣列的情況。

    # 隨機打亂陣列

    從陣列中隨機取出一個並刪去，重覆至陣列取完

    var result = [];
var arr = [1, 2, 3, 4, 5];
//將陣列長度存起來，以確保迴圈執行次數
var length = arr.length;
for(var i = 0; i < length; i++) {
	//arr.length 因陣列內容被刪除而變動，
	var random = Math.floor(Math.random() * arr.length);
	result.push(arr[random]);
	arr.splice(random, 1); 
}
console.log(result);

    Fisher–Yates Shuffle 是目前效率最高的演算法。以下以 for 迴圈改寫比較容易理解。

    var arr = [1, 2, 3, 4, 5];
var length = arr.length;
var temp;
//迴圈執行 arr.length 次 (此例為 5 次)
for(var i = 0; i < arr.length; i++) {
	//隨機取出 0-4 間的整數
	var random = Math.floor(Math.random() * length);
	//減 1 是因為陣列最後一個的索引是陣列長度減 1
	length = length - 1;

	//兩兩交換，將陣列最後一個的值存在 temp 變數中
	temp = arr[length];
	//隨機選出的那個的值存在最後一個
	arr[length] = arr[random];
	//將 temp 的值存回隨機取出的那個
	arr[random] = temp;
}
console.log(arr);
