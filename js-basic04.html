    # JavaScript 基礎 IV

    # 函式／函數 (Function)

    將重覆使用的程式打包，並且可依代入的參數執行相同的程式，產生不同的結果。

    //宣告函式
function 函式名(參數1, 參數2, ..., 參數n) {
	函式執行的程式
}

    函式宣告之後如果要使用就在「函式名」後方加上括號，如果有參數，則依序加入括號中。

    //呼叫函式
函式名(參數1, 參數2, ..., 參數n);

    console.log(), alert(), document.write() 這些有帶括號就代表是函式，(執行環境內建的)。

    例如：將傳入的參數在 alert 訊息視窗中顯示

    function hello(str) {
	alert('傳入的參數是: ' + str);
}
hello('Just Do It');

    函式預設會有回傳值 (計算結果)，使用 return 告知回傳值是什麼，如果省略就是 undefined。

    在函式中執行到 return 時即跳出函式，不會再執行函式內其他的程式。

    例如：矩形的面積計算 = 長 x 寬

    function area(width, height) {
	return width * height;
}
var box = area(10, 10);
alert('面積是: ' + box + ' 平方單位');

    函式也可以指定給變數或是物件的屬性

    var area = function(width, height) {
	return width * height;
}
var box = area(20, 20);
alert('面積是: ' + box + ' 平方單位');

    <div id="btn">我是按鈕</div>
<script>
document.querySelector('#btn').onclick = function() {
	alert('滑鼠按下了！');
}
</script>

    沒有函式名稱的函式又稱做「匿名函式」

    多個要素共用一個函式，需要修改的時候就簡單多了

    <div id="btn-1">按鈕-01</div>
<div id="btn-2">按鈕-02</div>
<div id="btn-3">按鈕-03</div>
<div id="btn-4">按鈕-04</div>
<div id="btn-5">按鈕-05</div>
<script>
function clickMe() {
	alert(this.innerHTML);
}
for(var i = 1; i <= 5; i++) {
	var btn = document.querySelector('#btn-' + i);
	btn.onclick = clickMe; //注意！為什麼這裡不需要加括號？
}
</script>
    # 實作練習

    請撰寫一個函式，可以計算 a-b 之間的數的連乘積

    function sum(a, b) {
	var result = 1;
	for(var i = a; i <= b; i++) {
		result = result * i;
	}
	return result
}
sum(10, 20);
