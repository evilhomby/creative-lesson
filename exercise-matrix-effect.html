    # 關於 Matrix Effect 實作流程

    # 產生一行 (Line)

    • 隨機選出一個字元
    • 重覆 n 次隨機選出一個字元
    • 將 n 個隨機字元加到網頁上

    # 產生多行

    • 將前述「產生一行」做為原型
    • 重覆 m 次隨機建立實例 (實體) 並加到畫面上
    • 調整每行的 top, left 位置

    # 課程範例 (範例)

    程式碼是由上而下逐行執行；但開發的順序會依需求、思考模式…等跳來跳去。 練習時請自行思考，並比對程式碼理解為什麼是這樣的設計模式。(沒有標準答案或模式，適用即可)

    # 效果調整 (範例)

    • 使用 @keyframes 與 animation 建立動畫效果
    • 於 JavaScript 中控制 animationDelay, animationDuration 改變動畫延遲與持續時間。
    • 每行的水平位移可使用 window.innerWidth 取得瀏覽器寬度 (不含捲軸)
    • 使用 overflow:hidden 隱藏超出的部分，因此不會產生捲軸
    • 使用 CSS filter:blur(npx); 來產生模糊

    # 最後一碼隨機效果 (範例)

    • 使用 animtaion 的 steps(n) 來產生逐格動畫效果
    • 將隨機字串進行位移

    # 請自行參考上述步驟與最終效果，自行調整出滿意的成果。
