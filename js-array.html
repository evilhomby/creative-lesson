    # 關於 JavaScript 的陣列

    陣列可以想像成許多有號碼的抽屜，用來存放資料。使用時需要用號碼 (索引) 取出資料，號碼從 0 開始！

    陣列用「中括號」表示，每一個抽屜間用逗號分隔。例：

    //三格的抽屜
var arr = ['a', 'b', 'c'];

//用號碼 (索引) 取得抽屜裡的資料
console.log(arr[0]); // a

    陣列通常會搭配迴圈使用，例：

    //逐個列出陣列裡的資料
var arr = ['a', 'b', 'c'];
for(var i = 0; i < arr.length; i++) {
	console.log(arr[i]);
}

    使用上很常出現二維陣列 (就是陣列裡面放陣列的意思)。也可能出現三維、四維…等等，但實務上不常出現，因為很容易搞混。例如：

    //宣告一個二維陣列
var arr = [
	['a1', 'a2', 'a3'],
	['b1', 'b2', 'b3'],
	['c1', 'c2', 'c3']
];
//取出第二層的第二個 (b2)
console.log(arr[1][1]);

    二維陣列就需要跑兩次迴圈取出所有的資料，例：

    var arr = [
	['a1', 'a2', 'a3'],
	['b1', 'b2', 'b3'],
	['c1', 'c2', 'c3'],
	['d1', 'd2', 'd3']
];
for(var i = 0; i < arr.length; i++) {
	for(var j = 0; j < arr[i].length; j++) {
		console.log(arr[i][j]);
	}
}

    想看看 arr.length 與 arr[i].length 是多少呢？ 此例 arr.length = 4 而 arr[i].length = 3

    # document.querySelectorAll() 的兩種輪循方法

    實作上建議用第二種方式較為簡潔。

    <div class="btn">btn-1</div>
<div class="btn">btn-2</div>
<div class="btn">btn-3</div>
<div class="btn">btn-4</div>
<div class="btn">btn-5</div>

<script>
var btns = document.querySelectorAll('.btn');

//第一種方法
for(var i = 0; i < btns.length; i++) {
	btns[i].onclick = function(){
		alert('clicked');
	}
}

//第二種方法
btns.forEach(function(v, i) {
	btns[i].onclick = function() {
		alert('clicked');
	}
});
</script>

    # 一維陣列轉為無序清單 <ul> 並顯示在網頁上。
    <html>
<body>
<div id="wrapper"></div>
<script>
var members = [
	'maureen',
	'leah',
	'jojo',
	'bora',
	'elaine'
];
var html = '';
html += '<ul>';
for(var i = 0; i < members.length; i++) {
	html += '<li>' + members[i] + '</li>';
}
html += '</ul>';
document.querySelector('#wrapper').innerHTML = html;
</script>
</body>
</html>
    # 二維陣列轉為表格並顯示在網頁上。
    <html>
<body>
<div id="wrapper"></div>
<script>
var arr = [
	['a1', 'a2', 'a3'],
	['b1', 'b2', 'b3', 'b4'],
	['c1', 'c2', 'c3', 'c4', 'c5'],
	['d1', 'd2', 'd3', 'd4', 'd5', 'd6']
];
var html = '';
html += '<table border="1">';
for(var i = 0; i < arr.length; i++) {
	html += '<tr>';
	for(var j = 0; j < arr[i].length; j++) {
		html += '<td>' + arr[i][j] + '</td>';
	}
	html += '</tr>';
}
html += '</table>';
document.querySelector('#wrapper').innerHTML = html;
</script>
</body>
</html>
