    # JavaScript 基礎

    程式基礎通常包含以下：

    • 資料型態、變數與常數
    • 條件判斷
    • 迴圈
    • 函數

    # JavaScript 的資料型態

    資料型態也就是我們說的類別、型別，分基本 (primitive) 和複合 (composite) 大類

    # 基本資料型態：

    # String 字串

    用引號包起來，一般來說優先使用單引號，將雙引號留給 HTML 標記使用。

    //使用單引號包起來的字串
var example = 'this is a example';

//使用雙引號包起來的字串
var example = "this is a example";

//多行文字
var example = `
line1
line2
`;
    # Number 數值

    包含整數、浮點數 (有小數點的數值) 和幾個特殊值。

    • 最大值：Number.MAX_VALUE
    • 最小值：Number.MIN_VALUE
    • 正無限大：Number.POSTIVE_INFINITY
    • 負無限大：Number.NAGTIVE_INFINITY
    • 非數值：Number.NaN
    //整數
var example = 1;

//浮點數
var example = 1.234
    # boolean 布林

    所謂的真偽，值只有 True 與 False 兩種，搭配條件判斷使用。

    False 家族：(0, NaN, '', null, undefined) 做型別轉換時為 False

    var example = true;

    比較有分「一般相等 (自動做型別轉換後比較)」與「嚴格相等 (不做型別轉換)」

    //一般相等，0　做型別轉換為 false，兩者相等得到 true
0 == false;

//嚴格相等，不做型別轉換，兩者型別不同得到 false
0 === false;
    # null 空物件

    宣告一個什麼都沒有的變數時使用。

    var example = null;
    # undefined 未定義

    變數宣告後未給值即為未定義

    var example;
    # 複合資料

    基本資料型態之外的都可視為是 Object，常用的有：

    • array 陣列：生活化的說法即為用數字為索引的抽屜
    • object 物件：用名稱為索引的抽屜
    • Math 數學
    • Date 日期

    # 變數 (variable) 是什麼？

    簡單的說變數就是一個裝資料的容器，有以下的特性：

    • JavaScript 是一種弱型別的程式語言，因此不需定義變數的資料型態。 例：
    var example = 123; //此時為數值
example = 'abc'; //資料改為字串
    • 宣告變數使用關鍵字 var 或 let (它們之間有差異)
    • 沒有使用 var 宣告的變數會成為全域變數，請養成用 var 宣告的好習慣。
    • 全域變數任何地方都可以使用
    • 區域變數僅在特定區域才能使用
    • 變數宣告後若無賦值，即為 undefined (未定義)，例：
    var example;
console.log(example); //得到 undefined

    宣告變數可以解讀為：我要在眾多抽屜中找一個出來，給予一個名稱方便稍候用來存放與讀取資料。 賦值是指將等號右邊的資料放到抽屜裡。

    @Jojo: 全域變數與區域變數用「護照」與「身份證」為例子補充

    # 常數是什麼？

    常數就是不能改變的資料，宣告時就必須給值，例如：光速、圓周率…等，使用 const 來宣告。 (改了會怎麼樣？會發生錯誤)

    const pi = 3.1415;

    # 變數的使用實例

    # 實例一、假設我們要進行一個很簡單的計算，例：x + y = z。程式會這樣寫：
    var x;
var y;
var z = x + y;

    但這樣 x, y 沒有給值會是 undefined，而 z 會是 NaN。修改一下：

    var x = 1;
var y = 2;
var z = x + y; // z = 3

    實作上會把上例做成函式來應用，代入 x, y 得到不同的 z。例：

    function sum(x, y) {
	var z = x + y;
	return z;
}
var ans = sum(1, 2);
    # 實例二：用變數記錄重覆使用的東西

    選擇頁面上的要素，在 CSS 是使用選擇器 (Selector)，例如：

    <style>
#example {
	background: #f00;
}
</style>

<div id="example">紅色背景</div>

    而 JavaScript 借用了 CSS 的選擇器規則來存取，例如：

    <div id="example">紅色背景</div>
<script>
document.querySelector('#example').onclick = function() {
	alert('紅色背景被點擊了！');
}
</script>

    延伸問題：有沒有發現 <script> 寫在被選擇的要素下方而不是上方？若寫在上方會怎麼樣呢？

    在前面的例子中，我們使用 JS 選擇到了要執行動作的要素，若要重覆使用可以用以下的方式：

    <script>
document.querySelector('#example').onclick = function() {
	alert('紅色背景被點擊了！');
}
document.querySelector('#example').ondblclick = function() {
	alert('紅色背景被 Double Click 了！');
}
document.querySelector('#example').onmouseover = function() {
	alert('滑鼠滑過紅色背景！');
}
</script>

    一直有重覆的部分出現，倘若要素的 id 屬性修改了，開發者就要逐個修改。而聰明的作法則是將重覆的部分存在變數之中

    <script>
var div = document.querySelector('#example');
div.onclick = function() {
	alert('紅色背景被點擊了！');
}
div.ondblclick = function() {
	alert('紅色背景被 Double Click 了！');
}
div.onmouseover = function() {
	alert('滑鼠滑過紅色背景！');
}
</script>
