    # jQuery Slider 函式庫: Owl Carousel 2

    課程推薦使用的是 Owl Carousel 2，以 CDNJS 為載入來源，實作時可自行下載到電腦中。

    # 使用方式

    • 載入 owl.carousel.css，而 owl.theme.carousel.css 可自行選擇性是否載入
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/OwlCarousel2/2.3.4/assets/owl.carousel.min.css" />
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/OwlCarousel2/2.3.4/assets/owl.theme.default.min.css" />
    • 載入 jquery-1.8.3 以上版本與 owl.carousel.js
    <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/OwlCarousel2/2.3.4/owl.carousel.min.js"></script>
    • 建立 HTML 結構如下，如果有使用 owl.theme.carousel.css 請在類別加上 owl-theme
    <div class="owl-carousel owl-theme">
	<div class="item">A</div>
	<div class="item">B</div>
	<div class="item">C</div>
	<div class="item">D</div>
	<div class="item">E</div>
	<div class="item">F</div>
	<div class="item">G</div>
	<div class="item">H</div>
	<div class="item">I</div>
</div>
    • 啟用 owlcarousel
    $(function() {
	$('.owl-carousel').owlCarousel();
});
